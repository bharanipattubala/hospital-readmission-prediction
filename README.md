# 🏥 Hospital Readmission Prediction
### End-to-End Data Analytics Project | SQL • Python • Machine Learning • Power BI

---

## 📌 The Business Problem

Hospitals across the world lose millions every year due to unplanned 
patient readmissions within 30 days of discharge.

The real question is: **Can we predict which patients will come back 
— before they even leave?**

This project answers exactly that. Using 100,000+ real patient records 
from 130 US hospitals, I built a complete analytics solution that 
identifies high-risk patients and gives hospital management 
clear, actionable recommendations.
